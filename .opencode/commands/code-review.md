# Comprehensive Code Review

Perform a multi-layer code review using parallel agents:

1. **code-quality-enforcer**: Check standards, complexity, duplication
2. **testing-qa-expert**: Review test coverage and quality
3. **accessibility-expert**: Audit WCAG compliance
4. **superpowers-code-reviewer**: Review against project plan

Review checklist:
- TypeScript strict mode compliance
- Proper error handling
- Test coverage > 80%
- Accessibility (labels, contrast, keyboard nav)
- Performance (bundle impact, re-renders)
- Security (input validation, auth checks)
- Documentation (comments, README updates)

Output a structured review report with:
- Summary table of issues by category
- Specific code examples
- Actionable recommendations
- Priority rankings
