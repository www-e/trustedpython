# Performance Optimization Check

Analyze and optimize application performance using parallel agents:

1. **perf-analyzer**: Profile Core Web Vitals and bundle size
2. **prisma-optimizer**: Review slow database queries
3. **react-specialist**: Check for unnecessary re-renders

Steps:
- Run `pnpm run build:analyze` to get bundle report
- Check database query logs for N+1 issues
- Review React component rendering patterns
- Provide prioritized optimization recommendations

Format output as a visual task summary with emoji status indicators.
