
User screens
This is the main app experience for the regular user role, which acts as both buyer and seller. The same person can browse listings, create listings, request deals, chat, and view history.

Home screen — Featured listings, latest listings, categories by game, quick filters, search bar.

Search screen — Search by game, title, price, rank, server, platform, seller, or keyword.

Filter / sort screen — Filter by game, price range, account level, verified seller, mediator-friendly listings, newest, oldest, cheapest.

Category listings screen — All accounts inside one game category such as PUBG or Free Fire.

Listing details screen — Full account post with title, price, game details, screenshots, description, seller info, status, and request-to-buy CTA.

Full-screen gallery screen — Opens screenshots and account images in detail.

Seller public profile screen — Seller info, active listings, sold history, trust indicators, ratings if you add them later.

Create listing screen — Seller enters account title, game, description, price, tags, account details, and uploads screenshots.

Edit listing screen — Update listing data, replace screenshots, change status, or archive listing.

Listing preview screen — Shows the seller exactly how the post will appear before publishing.

My listings screen — Tabs for active, pending, sold, archived.

Request to buy screen — Buyer confirms interest in a listing and starts the transaction flow.

Choose mediator screen — Buyer sees available mediators with badge, tier, trust level, limits, online status, and short intro.

Request submitted screen — Confirmation that seller and mediator were notified and chats were created.

Notifications center — Deal requests, deal accepted, new messages, proof uploaded, deal closed, listing sold.

Chat inbox screen — All active chats and closed chats, grouped by role and deal status.

Group deal chat screen — Main chat for buyer, seller, and mediator together.

Private mediator chat screen — Template used for buyer-mediator or seller-mediator private chat.

Deal details screen — Shows listing summary, participants, selected mediator, timestamps, and current deal status.

Upload proof screen — Upload screenshot or image proof inside a deal flow.

Sold page — Public or semi-public page showing sold accounts for trust and marketplace activity.

My history screen — User’s completed deals, canceled deals, closed chats, and sold/bought records.

Saved listings screen — Bookmarked listings the user wants to revisit.

User profile screen — Avatar, bio, trust level, sold count, bought count, active listings.

Edit profile screen — Update name, avatar, bio, contact preferences, and privacy settings.

Settings screen — Password, notifications, language, blocked users, help, logout.

Help / support screen — FAQs, report problem, safety tips, contact admin.

Report listing / report user screen — Submit abuse, scam, fake screenshots, or suspicious behavior.
