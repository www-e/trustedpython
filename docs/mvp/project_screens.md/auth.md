
Auth screens
These are the screens every regular user needs before entering the marketplace. Mediators do not sign up themselves, so their auth flow is simpler.

Splash screen — App logo, short brand statement, loading state, auto-route to onboarding or home.

Onboarding screen 1 — Explains the app concept: buy and sell gaming accounts safely through mediators.

Onboarding screen 2 — Explains how deals work: choose mediator, open chats, exchange proof, close deal.

Onboarding screen 3 — Explains trust and safety rules, banned behavior, and account verification expectations.

Login screen — Email/phone and password login, social login if needed, link to sign up and forgot password.

Sign up screen — Create a regular user account, basic info like name, phone, email, password, country.

OTP / verification screen — Verify email or phone before full access.

Forgot password screen — Start password reset flow.

Reset password screen — Create a new password.

Complete profile screen — Add avatar, username, short bio, preferred games, and optional trust info.

Notification permission screen — Ask for push notification access.

Terms and safety acceptance screen — User accepts marketplace rules and risk disclaimers.
