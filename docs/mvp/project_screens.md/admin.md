
Admin screens
The admin side is best designed as a web dashboard, even if the marketplace is mobile-first. This area controls trust, moderation, mediator creation, and marketplace oversight.

Admin login screen — Secure admin authentication.

Admin dashboard screen — Top-level KPIs such as users, mediators, active deals, sold listings, flagged content, and bans.

Users management screen — View all regular users, search, filter, suspend, ban, or inspect.

User details screen — Full profile, active listings, history, reports, and moderation actions.

Mediators management screen — View all mediators, their tiers, performance, assistants, and status.

Create mediator account screen — Admin manually creates mediator account and initial credentials.

Edit mediator screen — Update badge, tier, permissions, limits, assistants, and account status.

Tier management screen — Define tier names and limits such as contribution count, badge style, assistant count, reopen-chat permission, and visibility perks.

Listings management screen — View all listings with filters by game, seller, status, and flagged state.

Listing details moderation screen — Inspect screenshots, post content, reports, and remove or hide listing.

Active deals monitoring screen — See all ongoing deals and their current stage.

Closed deals monitoring screen — Review finished deals for auditing and support.

Sold accounts screen — Marketplace sold archive.

Reports / abuse screen — Central place for scam reports, fake proof claims, and moderation review.

Ban management screen — Manage banned users and mediators, reasons, and durations.

Notifications / announcements screen — Send system-wide or role-specific notifications.

Admin settings screen — Security, roles, access control, and platform configuration.
